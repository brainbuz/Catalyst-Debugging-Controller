NAME

TestApp::Controller::Root - Root Controller for TestApp

VERSION 1.04

DESCRIPTION

This is a small application useful for configuration testing. It can tell you what time it is. The useful thing it does is dump out lots of information about your environment.

METHODS

index

The root page (/) This is the Catalyst Default root page. Your application is able to produce output.

brief

Will tell you who is logged in and what time it is.

spew

Dumps a whole bunch of stuff out of your environment. Like pages of it.

form

If you point a form at this page instead of spew, it will only tell you the GET and POST parameters. spew already includes the form data dump, use it to get everything.

Deployment Within a Catalyst Application

  •  Rename this controller Diag.pm (or whatever suits you).
  •  Change the Package Name line to match your application.
  •  Rename sub index to sub diag :Path('/diag') :Args() {
  •  Delete the subs "end" and "default".
  •  Adjust the paths to the subs form, spew, and brief.

default

Standard 404 error page

end

Attempt to render a view, if needed.
